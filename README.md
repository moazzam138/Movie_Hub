🎬 MovieHub

MovieHub is designed for movie enthusiasts who want to discover and explore films from around the world. The application integrates with The Movie Database (TMDb) API to provide real-time movie data, including posters, ratings, descriptions, cast information, and similar movie recommendations.

The project demonstrates modern frontend development concepts such as API integration, component-based architecture, state management, routing, and responsive UI design.

✨ Features:

🔍 Movie Search
-Search movies by title.
-Instant access to movie information.
-Dynamic search results powered by TMDb API.

📈 Trending Movies
-Display daily trending movies.
-Automatically fetches the latest popular content.

🎭 Movie Details
-Detailed movie information page.
-Movie poster and backdrop.
-Release date.
-Ratings and popularity score.
-Movie overview and description.

👥 Cast Information
-View the cast members of a selected movie.
-Displays actor names and related information.

🎬 Similar Movies
-Get recommendations based on the selected movie.
-Helps users discover related content.

📱 Responsive Design
-Fully responsive interface.
-Optimized for desktop, tablet, and mobile devices.

⚡ Fast Performance
-Built using React and Vite for faster development and performance.
-Efficient API calls using Axios.


| Technology       | Purpose            |
| ---------------- | ------------------ |
| React.js         | Frontend Framework |
| Vite             | Build Tool         |
| Tailwind CSS     | Styling            |
| React Router DOM | Routing            |
| Axios            | API Requests       |
| TMDb API         | Movie Data Source  |
